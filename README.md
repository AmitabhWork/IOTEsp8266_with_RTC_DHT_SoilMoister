# IOTEsp8266_with_RTC_DHT_SoilMoister

Connecting DHT22, RTC_DS1307, soilmoister sensor with ESP8266 AND getting the data.

//Connection

## RTC - ESP8266

SDA - D2
SCL - D1
GND - GND
VCC - 3.3V

## DHT22 - ESP8266

OUT/DATA - D3
GND - GND
VCC - 3.3V

## SoilMoister -ESP8266

AO - A0
GND - GND
VCC - 3.3V
